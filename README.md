# 🛍️ OOP Ecommerce

Учебный проект для отработки объектно-ориентированного программирования на примере интернет-магазина.

---

## 📦 Классы

### 📱 Product (Товар)

| Атрибут | Тип | Описание |
|---------|------|-------------|
| `name` | `str` | Название товара |
| `description` | `str` | Описание товара |
| `price` | `float` | Цена товара |
| `quantity` | `int` | Количество в наличии |

### 📂 Category (Категория)

**Атрибуты экземпляра:**

| Атрибут | Тип | Описание |
|---------|------|-------------|
| `name` | `str` | Название категории |
| `description` | `str` | Описание категории |
| `products` | `List[Product]` | Список товаров в категории |

**Атрибуты класса:**

| Атрибут | Тип | Описание |
|---------|------|-------------|
| `category_count` | `int` | Общее количество категорий |
| `product_count` | `int` | Общее количество товаров |

## 🧪 Тестирование

```
# Запуск всех тестов
pytest tests/ -v

# Проверка стиля кода
flake8 src/ tests/
```

## 🚀 Установка и запуск
```
# Установка зависимостей
poetry install

# Активация виртуального окружения
poetry shell

# Запуск тестов
pytest tests/ -v
```

